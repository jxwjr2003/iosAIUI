# 增强模拟服务器和代理服务器功能

## 目标
扩展 LocalProxy 的现有功能，添加完整的模拟服务器和增强的代理服务器能力，为客户端测试提供更强大的工具支持。

## 问题陈述
当前 LocalProxy 的模拟服务器功能有限，缺乏配置持久化、实时日志监控和错误模拟能力。代理服务器功能也需要增强以支持更完整的请求转发和监控。

## 解决方案概述
1. **模拟服务器增强**：
   - 图形界面配置本地端口、HTTP方法、API路径、头部、JSON body
   - 配置持久化存储，支持命名配置和列表管理
   - 实时展示客户端请求和服务器响应的日志
   - 支持手动复制JSON文本
   - 预定义错误码模拟（404, 500）

2. **代理服务器增强**：
   - 配置本地代理端口和目标服务器地址
   - 完整的请求/响应数据转发
   - 实时日志监控客户端请求
   - 支持HTTP和HTTPS协议

3. **用户体验**：
   - 图形界面操作
   - 日志支持过滤和搜索
   - 支持最多3个并发请求

## 成功标准
- 用户能够创建、保存和加载模拟服务器配置
- 实时显示请求和响应日志，支持过滤搜索
- 能够模拟404和500错误响应
- 代理服务器正确转发请求并记录日志
- 支持HTTP和HTTPS协议
